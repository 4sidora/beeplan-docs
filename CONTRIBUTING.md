# Contributing to BeePlan

Проект разбит на несколько репозиториев; **общая** документация и этот файл — в репозитории **beeplan-docs**.

## Репозитории

| Репозиторий | Изменения в коде / контракте |
|-------------|------------------------------|
| beeplan-api | REST, OpenAPI (`docs/openapi.yaml`), миграции БД |
| beeplan-web | UI, вызовы API |
| beeplan-edge | Прошивка улья, бинарный протокол ESP‑NOW |
| beeplan-gateway | Прошивка концентратора, приём ESP‑NOW, uplink |

Сквозные правки (архитектура, железо, требования) — в **beeplan-docs** в том же PR/коммите, где меняется поведение, если применимо.

## Pull requests

- Опишите, что меняется и зачем.
- По возможности приложите скриншот UI или пример запроса/ответа API.

## Код

- Python: `ruff`/`black` по мере подключения в CI (пока необязательно).
- TypeScript: `npm run build` без ошибок.
- Прошивки: не коммитьте пароли Wi‑Fi и токены — используйте `config.h` только локально или шаблон `config.example.h` (когда появится в репозитории).
